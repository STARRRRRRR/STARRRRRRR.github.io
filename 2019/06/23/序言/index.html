发表于 2019-06-23 | 分类于 数据结构与算法

因为好久没用Go来写东西了，所以打算借着学习数据结构与算法来重温一下Go语言。在这里想分享一下自己打算怎么学Go的数据结构与算法。

1.经典著作镇楼

当然，这两本书我都不是拿来看的。有事没事烧香拜一拜，说不定期末考就过了~

个人不推荐上面两本书，虽然几十年来无数大牛强力推荐，但是作为一只菜鸟，我可以很负责任地说体验效果极差！！

为啥？因为实在是太丑了！！（来自颜狗的呐喊！

不得不说机械工业出版社简直就是直男一般的审美
这套黑皮书的封面丑的一批，本颜狗一点看的欲望都没有

还有算法这本书
如此令人窒息的排版和字体
所以说书还是要看自己的倾向啊，别人说好不一定就适合自己
接下来介绍一下适合像我这样的菜鸟的算法书

2.入门书籍

第一当然是《图解算法》啦！！！

通俗易懂地用漫画解释基础概念和算法，虽然是用python实现的，但是很适合初学者理解。可以像我一样看漫画跳过算法代码，几天就能看完，非常优秀的书！

第二没有前面的书有名，但我个人认为在某些程度这本书做的更好

看这个阿三作者
是不是感觉迎面一股浓浓的咖喱味~

话说回来，虽然阿三在军事工业上天天摔飞机炸航母沉潜艇

但是人家其他方面还是很靠谱滴，这本书很清楚地阐明了各种概念，用Java语言实现，例题也很多，最重要的是，人家排版好看啊！！
图书馆就有这本书，有兴趣的可以去图书馆看看。
